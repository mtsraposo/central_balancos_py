1. Instalar Ruby 2.5.7
2. Configurar no Google Cloud API, uma credencial OAUTH Client ID > Web Application.
3. Preencher o arquivo config.json com o client_id e client_secret da Google Cloud API da sua conta.
4. No command prompt (cmd), rodar `gem install bundler`.
5. Procure o caminho absoluto do arquivo script.rb, e.g. \Users\<...>\lib\script.rb. <...> é o caminho que você encontrará no seu PC.
6. No command prompt (cmd), execute `cd \Users\<...>\lib\`.
7. No command prompt (cmd), execute `bundle install`.
8. Crie uma planilha com o nome `CENTRAL` e salve no Drive da conta que gerou as credenciais no passo 2. Certifique-se de que existe somente uma planilha com esse nome no Drive.
9. No command prompt (cmd), execute `ruby lib\script.rb`.
10. Entre 1 e deixe rodar para atualizar os CNPJs.
11. No command prompt (cmd), execute `ruby lib\script.rb`.
12. Entre 3 e deixe rodar para preencher a planilha.
